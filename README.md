# Noda API

## Project Summary
The Noda API is a backend RESTful service developed with Spring Boot 3 and Java 21. It implements a layered architecture to manage user data, focusing on separation of concerns and maintainability. This project serves as a technical foundation for mastering enterprise Java standards and RESTful API design.

## Technical Stack
- Java 21
- Spring Boot 3.4.3 (Web, Data JPA, Test)
- PostgreSQL
- Project Lombok
- Maven

## Architecture
- **Controller Layer**: Manages REST endpoints and HTTP request/response flow.
- **Service Layer**: Handles business logic and service orchestration.
- **Model Layer**: Defines data entities and utilizes Lombok for boilerplate reduction.

## Features
- Scalable layered architecture.
- Constructor-based dependency injection.
- Automated data object mapping.
- JSON-based communication.
